# ML Classification Assignment - Streamlit UI

This project implements 6 ML classification models using the UCI Breast Cancer dataset and provides a
Streamlit UI.

## Models Implemented
1. Logistic Regression
2. Decision Tree
3. KNN
4. Naive Bayes
5. Random Forest
6. XGBoost

## Metrics
- Accuracy
- AUC
- Precision
- Recall
- F1 Score
- MCC


## Run Instructions

1. Install dependencies
pip install -r requirements.txt

2. Run Streamlit Application.
streamlit run application.py

3. Open browser link shown below.
4. https://sujeet-yadav-bits-mlassignment2.streamlit.app/

App Run Steps:

Download the sample test dataset First.
Upload the downloaded  TES Data CSV file.
Select a classification model
Click Run Evaluation
Review metrics, confusion matrix, and classification report
